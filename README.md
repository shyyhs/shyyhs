# Hi, I'm Haiyue Song

I am an LLM Engineer at [Preferred Networks](https://www.preferred.jp/en/), working on natural language processing and large language models.

My research and engineering interests include machine translation, multilingual NLP, low-resource language technologies, document translation, and practical evaluation of language systems.

## Current Focus

- Large language models and NLP systems
- Machine translation and multilingual modeling
- Low-resource and under-represented language technologies
- Structured document translation and evaluation
